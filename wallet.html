MetaMask


It is a wallet where tokens can be bought, stored, sent and exchanged, available as an extension browser and with a mobile application. MetaMask equips you with a key vault, secure login to manage your digital assets.

It is used by more than 1M monthly active users.


MetaMask available for Desktop and App with IOS and Android operating systems.

Access for desktop


MetaMask has an extension for Google Chrome, Firefox, Opera, Brave and Edge browsers. Once you have added it, the fox icon will appear.


Click on the Metamask icon, read and accept the terms and conditions, click on create wallet, then create a secure password and finally, you it will show 12 words (recovery seed), which you must save and write down, without them you will not be able to recover your wallet in case of loss. And with these steps you have created an account. Below that is a "desktop" download button. For mobile you must go to your application provider.



Access for movil


You must download the app from your mobile, once the application is downloaded select "create a new wallet", create a password and then it will show you 12 words (recovery seeds), you must copy in the order that was assigned to you, keeping it in a safe place, finally it will ask you to confirm each phrase according to the order provided and you will have a Metamask account.


Setting up wallet



Main Net


Binance Smart Chain network setup. To configure this network where we are going to operate, we must select the settings menu icon then press networks and it will give us the option to add a network, then fill in the requested fields.

Network Name: Binance Smart Chain

New RPC URL: https://bsc-dataseed.binance.org/

String ID: 56

Symbol (optional): BNB

Block Explorer URL (optional): https://bscscan.com


You must have all these data to be able to configure your wallet to the network Binance Smart Chain.
